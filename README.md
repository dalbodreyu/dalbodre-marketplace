## 🚀 사용 방법 (Claude Code)

### 옵션 A — CLI로 빠르게

```bash
# 1단계: 이 마켓플레이스를 Claude Code에 등록 (한 번만 실행)
claude plugin marketplace add dalbodreyu/dalbodre-marketplace

# 2단계: 원하는 플러그인 설치
claude plugin install image2-creator-suite
```

### 옵션 B — TUI로 시각적으로

```
> /plugins
```

→ **Marketplaces** 탭 → "Add marketplace" → `dalbodreyu/dalbodre-marketplace` 입력  
→ **Discover** 탭 → 원하는 플러그인 선택 → **Space** 키로 설치
